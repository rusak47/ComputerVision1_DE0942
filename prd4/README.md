#### 3D punktu mākoņa iegūšana

**Mērķis:**

Apgūt dziļuma informācijas ieguvi no stereo pāriem un izveidot savu programmu 3D punktu mākoņu ģenerēšanai no disparitātes kartes.

**Uzdevumi:**

<br />

1. No Middlebury kalibrēta stereo pāra iegūt 3D punktu mākoni.
2. No pašu uzņemta stereo pāra (ar savu kalibrāciju) iegūt 3D punktu mākoni.
3. Abus mākoņus vizualizēt MeshLab (atsevišķi).

* <br />

**Rīki:**\
Python, OpenCV, Matplotlib, Meshlab

**Sagaidāmais rezultāts:**

<br />

* Programmas kods (.ipynb)
* Oriģinālie Middlebury datu kopas attēli (stereo pāris), disparitātes karte, ekrānattēli no Meshlab.
* Savi attēli (stereo pāris), disparitātes karte, ekrānattēli no Meshlab.
* Secinājumi (var tikt pievienoti teksta laukā, darba iesniegšanas uzdevumi, vai kā atsēvišķš pdf fails)
